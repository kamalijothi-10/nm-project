# nm-project
To supply leftover food to poor

---

## Implementation
1. Created **Custom Objects** in Salesforce for Donors, NGOs, and Volunteers.  
2. Designed **Lightning Pages** for user interaction.  
3. Implemented **Apex Controllers** to handle business logic.  
4. Added **Automation Rules** for notifications and status tracking.  
5. Used **Reports & Dashboards** to monitor donation statistics.
